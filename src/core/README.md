# Core Module

Core analysis engine and shared utilities.

## Components

- `pipeline/` - Analysis pipeline orchestration and stages
- `config/` - Configuration types and validation
- `dependency/` - Dependency graph analysis
- `partitioning/` - Code partitioning algorithms
- `ast_service.rs` - Shared AST parsing service
- `scoring.rs` - Code quality scoring
- `featureset.rs` - Feature extraction framework
- `errors.rs` - Error types

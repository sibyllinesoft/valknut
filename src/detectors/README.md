# Detectors

Analysis detectors for various code quality dimensions.

## Available Detectors

- `complexity/` - Cyclomatic and cognitive complexity analysis
- `structure/` - File and directory structure analysis
- `lsh/` - Locality-sensitive hashing for clone detection
- `coverage/` - Test coverage analysis
- `refactoring/` - Refactoring opportunity detection
- `cohesion/` - Semantic cohesion analysis
- `graph/` - Dependency graph metrics

# Source Code

Main source code for the valknut code analysis tool.

## Structure

- `api/` - Public API types and result structures
- `bin/` - Binary entry points (CLI, MCP server)
- `core/` - Core analysis engine and utilities
- `detectors/` - Analysis detectors (complexity, structure, LSH, etc.)
- `io/` - I/O utilities (caching, reports)
- `lang/` - Language adapters (Python, Rust, TypeScript, Go, JavaScript)
- `oracle/` - AI oracle integration for analysis suggestions
- `doc_audit/` - Documentation audit utilities
